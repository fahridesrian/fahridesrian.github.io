About

A bit about my work

I'm a Mathematics undergraduate at Universitas Indonesia with a strong foundation in mathematical modeling, statistics, and machine learning. My primary interests lie in machine learning and cybersecurity, where I enjoy applying analytical thinking to solve real-world problems.

Throughout my academic journey, I have developed projects ranging from intelligent prediction systems and natural language processing to bioinformatics and network security. My current undergraduate thesis focuses on a hybrid proactive Network Intrusion Detection System that combines anomaly detection with time-series forecasting to improve early attack detection.

Beyond coursework, I enjoy exploring new technologies and expanding my technical skill set through personal projects. While Python is my primary language for machine learning and data science, I have also experimented with C++, CUDA for parallel computing, Rust, R, JavaScript with Vue.js, and MySQL across various academic and personal projects. I believe that choosing the right tool for a problem is just as important as building the solution itself.

Completed

PothRGBD: Pothole Detection with YOLOv8-Seg

A pothole detection and segmentation pipeline built on YOLOv8-Seg using the PothRGBD RGB-D dataset from Kaggle. The model was trained to accurately segment road damage from paired RGB and depth imagery, followed by depth-based post-processing for visualization and statistical analysis. Achieved a Mask mAP@50 of 97.20% and a Mask mAP@50-95 of 63.96%.

Completed

YOLO-SIBI: Real-Time Indonesian Sign Language Recognition

A real-time Indonesian Sign Language (SIBI) recognition system combining YOLOv8 hand detection with deep learning image classifiers. The application detects hand gestures from a webcam, classifies the corresponding alphabet, and displays prediction confidence in real time. DenseNet121 achieved the best classification accuracy of 89.88%.
